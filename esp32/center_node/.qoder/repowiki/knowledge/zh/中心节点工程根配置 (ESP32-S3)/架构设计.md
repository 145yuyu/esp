本模块为 leaf 工程根，仅包含跨构建系统的顶层配置文件：
- `CMakeLists.txt`：通过 `EXTRA_COMPONENT_DIRS` 将 `../common` 与 `components/` 注入 ESP-IDF 组件树，并 `include($ENV{IDF_PATH}/tools/cmake/project.cmake)` 以 `center_node` 作为项目名；依赖外部 IDF 工具链。
- `platformio.ini`：PlatformIO 侧的 Arduino 框架配置，目标板 `esp32-s3-devkitc-1`，通过 `lib_deps` 引入本地 `../common/TFT_eSPI` 与 Adafruit DHT 库，并通过大量 `-D` 宏（ST7735、引脚映射、SPI 频率等）裁剪 TFT_eSPI 行为。
- `sdkconfig.defaults`：ESP-IDF Kconfig 默认值，集中定义 Wi-Fi SSID/密码、ESP-NOW 最大加密数、FreeRTOS 时钟与日志级别、主任务栈大小等运行时参数。
- `.gitignore`：屏蔽 `.pio`、`build/`、`sdkconfig`、`*.bin/elf/map` 等生成产物及 VS Code 索引文件。
依赖方向单向向外：本目录不引用任何源码，仅由上层 CMake/PlatformIO 解析后组合 `common/` 与 `components/` 下的实际实现。
